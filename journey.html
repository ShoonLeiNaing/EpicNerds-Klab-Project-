LCD Touchscreen

Touchscreen မှာတော့ ပထမဆုံး Library ပြဿနာကြုံပါတယ် 🤧 ဒီမှာတော့ touchscreen တွေအတွက် နာမည်ကြီး library တွေထဲက UTFT library နဲ့ URTouch library ကို သုံးထားပါတယ် 😉 ကျွန်တော်တို့ သုံးထားတာကတော့ RobotDyn က TFT 2.8” LCD Touch screen module ပါ 😊 Expansion shield ကလည်း RobotDyn က ပါပဲ 🤗 Model အလိုက် code တွေမှာ modify လုပ်ရတဲ့ နေရာလေးတွေ ပါပါတယ်၊ အသေးစိတ်ကိုတော့ Library ရဲ့ official website မှာ ဖတ်ကြည့်နိုင်ပါတယ် 😁
